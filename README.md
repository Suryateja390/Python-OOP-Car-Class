# Python OOP - Car Class Example

This repository contains a basic Python program demonstrating core Object-Oriented Programming (OOP) concepts.

## Concepts Covered
- Class and Object
- Constructor (__init__)
- Instance variables
- Class methods
- Method calling through objects

## Program Features
The Car class contains attributes such as name, price, and color and includes methods to:
- Accelerate the car
- Display the car color
- Apply the brake system

This project is suitable for beginners who want to understand how OOP works in Python.
